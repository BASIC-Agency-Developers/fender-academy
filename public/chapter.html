About Electric Guitars

If you’ve never built a website before and you have no coding or design experience, this is the place to start. In this project, we learn how to build a modern portfolio website for desktops, tablets, and mobile devices. We start with basic HTML and CSS syntax. Next, we learn how to build custom web pages with an image gallery and contact page. Finally, we walk through how to share a website live on the web. PLO 70-80 words max.

What You’ll Learn
  • How to write basic HTML
  • How to style web pages with CSS
  • Learn about the Stratocaster history, features word
  • How to purchase a domain and hosting
